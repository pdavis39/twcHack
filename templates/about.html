Team Under the Weather

The motivation and technology for our hack.

Motivation

Athletes care about the weather

The purpose of this web site is to assist athletes as they prepare for their upcoming training by checking the daily weather and air quality.

Based on the temperature, air quality and precipitation the website provides custom feedback that includes motivational messages aligned to the weather and air quality conditions.

Applicaiton Logic

airQuality.py

This function calls AirNow's Zip Forecast to get current or historical forecasted AQI values and categories for a reporting area by Zip code.

app.py

Flask is a micro web framework written in Python and based on the Werkzeug toolkit and Jinja2 template engine. It is BSD licensed.

motivator.py

motivator takes output from airQuality.py, tendayForecast.py and quotes.py and customizes a user's weather report based on the air quality and weather features.

quotes.py

Proivdes customized quotes based on temperature , air quality, and preciptation.

tendayForecast.py

This function calls the weather API and based on the zip code provided by the user brings back weather information. Documenation for the API can be found at Weather Company Data for IBM Bluemix

zipToLatLong.py

This function akes a zip code input from the web application index.html page and returns a latitude and longitude used by other functions.
